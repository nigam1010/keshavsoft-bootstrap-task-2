Build Automation with Gulp

As part of the JAMstack architecture, we use Gulp.js for automating asset optimizations. This includes compressing images to ensure faster load times and improved performance.

Gulp Task: optimizeImages
  • Reads all images from src/images/
  • Optimizes them using gulp-imagemin
  • Outputs optimized images to dist/images/

Command used: npx gulp optimizeImages
